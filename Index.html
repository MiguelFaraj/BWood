📋 Step-by-Step Google Sheets Setup
Step 1 — Create your Google Sheet
Create a new Google Spreadsheet and add these sheet tabs (exact names matter):
PURCHASES STOCK_EXTRA PRIX PROJECTS USERS SUPPLIERS WORKMEN WORKERS_MP TASK_TEMPLATES
PURCHASES: id, date, month, code, category, type, supplier, qty, unit_price, total, status, notes
STOCK_EXTRA: date, project, code, category, type, quantity, unit_price, total, month
PRIX: code, category, type, price
PROJECTS: id, name, client, location, phone, created, active
USERS: id, username, password, fullname, role, whatsapp, created, active
SUPPLIERS: id, name, contact, phone, email, category, address, notes, active
WORKMEN: id, name, job, phone, salary, daily_wage, notes
MASTERPLAN: id, project, item_code, sub_type, material, etage, seq_num, unit_price, total, tasks, installation_member, installation_date, notes, created
  → tasks column: pipe-separated e.g. 0:CUTTING MELAMINE | 0:SANDING | 0:DELIVERY (0=pending, 1=done)
TASK_TEMPLATES: id, item_code, sub_type, tasks
  → tasks column: pipe-separated e.g. CUTTING MELAMINE | SANDING | PRESSING | DELIVERY TASK_TEMPLATES: id, item_code, sub_type, tasks
Step 2 — Create the Apps Script
In your Sheet, go to Extensions → Apps Script. Replace all code with this script:
const SHEET_ID = SpreadsheetApp.getActiveSpreadsheet().getId(); function doGet(e) { return handleRequest(e); } function doPost(e) { return handleRequest(e); } function handleRequest(e) { const p = e.parameter || {}; const body = e.postData ? JSON.parse(e.postData.contents || '{}') : {}; const action = p.action || body.action; const ss = SpreadsheetApp.openById(SHEET_ID); let result = {}; try { if (action === 'ping') { result = { ok: true, message: 'Connected!', sheets: ss.getSheets().map(s => s.getName()) }; } else if (action === 'read') { const sheetName = p.sheet || body.sheet; const ws = ss.getSheetByName(sheetName); if (!ws) { result = { ok: false, error: 'Sheet not found: ' + sheetName }; } else { const rows = ws.getDataRange().getValues(); result = { ok: true, data: rows }; } } else if (action === 'append') { const sheetName = body.sheet; const rows = body.rows || []; const ws = ss.getSheetByName(sheetName); if (!ws) { result = { ok: false, error: 'Sheet not found: ' + sheetName }; } else { rows.forEach(row => ws.appendRow(row)); result = { ok: true, appended: rows.length }; } } else if (action === 'overwrite') { const sheetName = body.sheet; const rows = body.rows || []; const ws = ss.getSheetByName(sheetName); if (!ws) { result = { ok: false, error: 'Sheet not found: ' + sheetName }; } else { ws.clearContents(); if (rows.length > 0) ws.getRange(1, 1, rows.length, rows[0].length).setValues(rows); result = { ok: true, written: rows.length }; } } else { result = { ok: false, error: 'Unknown action: ' + action }; } } catch(err) { result = { ok: false, error: err.toString() }; } return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); }
Step 3 — Deploy the Web App
  1. Click Deploy → New Deployment
  2. Select type: Web App
  3. Set "Execute as": Me
  4. Set "Who has access": Anyone
  5. Click Deploy and authorize permissions
  6. Copy the Web App URL — it ends with /exec
Step 4 — Connect & Sync
  1. Paste the Web App URL in the field above
  2. Click Save URL then Test Connection
  3. That's it — all data syncs automatically from this point on
⚠️ Re-deploy the Apps Script (Deploy → Manage Deployments → Edit → New Version) whenever you update the script code.
